== Description ==

当用户没有头像时,就显示一个可爱的Android机器人.

= prerequisite =
需要先安装[processing.js](https://wordpress.org/extend/plugins/processing-js/)

== Installation ==


1. install [processing.js](https://wordpress.org/extend/plugins/processing-js/)

2. unzip or put folder `imandroid` to the `/wp-content/plugins/` directory

3. Activate the plugin through the 'Plugins' menu in WordPress
